iPhoneSettings.screenCanDarken 屏幕可以变暗

static var screenCanDarken : bool

Description描述

Allows device to fall into "sleep" state with screen being dim if no touches occurred. Default value is true.

如果没有触动发生，允许设备变成"sleep(休眠)"的状态并且屏幕是暗淡的。默认值是true(真)。

It could be useful to set screenCanDarken to false for games which uses accelerometer as a main source of input from user. However such games should allow screen dimming while in menu or paused for better battery life.

如果重力感应是用户的主要输入源的话，那么设置screenCanDarken为false是非常有用的。但是，这样的游戏应该让屏幕变暗当在菜单里面或者暂停的时候，使得延长电池寿命

// Disable screen dimming
//禁用屏幕变暗
iPhoneSettings.screenCanDarken = false;
