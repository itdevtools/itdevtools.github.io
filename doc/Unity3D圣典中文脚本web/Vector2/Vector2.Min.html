Vector2.Min 最小

static function Min (lhs : Vector2, rhs : Vector2) : Vector2

Description描述

Returns a vector that is made from the smallest components of two vectors.

返回一个由两个向量的最小组件组成的向量。

参见：Max 函数.

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	public Vector2 a = new Vector2(1, 3);
	public Vector2 b = new Vector2(4, 2);
	public void Awake() {
		print(Vector2.Min(a, b));
	}
}
var a : Vector2 = Vector2 (1, 3);
var b : Vector2 = Vector2 (4, 2);
print (Vector2.Min(a,b)); // prints (1.0,2.0)
