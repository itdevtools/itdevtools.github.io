AndroidJavaObject.Get.<FieldType> 获取域类型

function Get.<FieldType> (fieldName : string) : FieldType

Parameters参数

  • fieldName
    The name of the field (e.g. int counter; would have fieldName = "counter")
    域的名称。

Description描述

Get the value of a field in an object (non-static).

获取一个对象上的域值（非静态）。

The generic parameter determines the field type.

一般参数定义域类型。

// Create a android.os.Message object, and retrieve the default value of the field 'what'.
//创建一个android.os.Message对象并检索域what的默认值

function Start() {
	var jo = new AndroidJavaObject("android.os.Message");
	var valueString = jo.Get.<int>("what");
}
