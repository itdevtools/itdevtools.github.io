Cloth.bendingStiffness 抗弯硬度

var bendingStiffness : float

Description描述

Bending stiffness of the cloth.

布料的抗弯硬度（坚挺的程度）

Must be between zero and one. Setting the value to zero disables bending stiffness simulation. In many cases, when you want a cloth to be more stiff, actually reducing the number of vertices will get you better results - and performance.

0 ~ 1 之间。设置为0关闭这个模拟属性。 在大多数情况下，当你想要一块比较硬的布料，实际上可以减少布料的顶点数，会得到更好的效果和性能。

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	public void Awake() {
		transform.GetComponent<Cloth>().bendingStiffness = 1;
	}
}
// Set the bending Stiffness to 1 regardless if the cloth
//设置布料的bending Stiffness为1
// is a Skinned Cloth or Interactive.
//可以是Skinned Cloth或者Interactive Cloth.
