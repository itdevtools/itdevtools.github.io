AudioChorusFilter.delay 延迟

var delay : float

Description描述

Chorus delay in ms. 0.1 to 100.0. Default = 40.0 ms.

以毫秒为单位合声延迟，0.1～100.0。默认=40.0ms。

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	void Start() {
		GetComponent<AudioChorusFilter>().delay = 0.1F;
	}
}
// Dont use delay on the filter.
//在这个滤波器上不使用延迟
@script RequireComponent(AudioSource)
@script RequireComponent(AudioChorusFilter)

function Start() {
	GetComponent(AudioChorusFilter).delay = 0.1;
}
