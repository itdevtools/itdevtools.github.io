MovieTexture 影片纹理

Inherits from Texture

Movie Textures are textures onto which movies are played back.

MovieTexture是可以在上面播放电影的纹理。

They can be used for cutscene movie sequences, or to render movies into the scene itself.

它们可以用于过场动画影片序列，或者渲染电影到场景中。

Variables变量

  • audioClip
    Returns the AudioClip belonging to the MovieTexture.
    返回属于MovieTexture的AudioClip。
  • loop
    Set this to true to make the movie loop.
    设置为true，将使影片循环播放。
  • isPlaying
    Returns whether the movie is playing or not
    返回影片是否正播放。
  • isReadyToPlay
    If the movie is downloading from a web site, this returns if enough data has been downloaded so playback should be able to start without interruptions.
    如果影片是从网站上下载的，这个返回是否已经下载了足够的数据以便能够不中断的播放它。
  • duration
    持续时间

Functions函数

  • Play
    Starts playing the movie.
    开始播放影片。
  • Stop
    Stops playing the movie, and rewinds it to the beginning
    停止播放影片，并后退到开始处。
  • Pause
    Pauses playing the movie.
    暂停播放影片。

Inherited members继承成员

Inherited Variables继承变量

  • width
    Width of the texture in pixels (Read Only)
    以像素为单位的贴图宽度 （只读）
  • height
    Height of the texture in pixels (Read Only)
    以像素为单位的贴图的高度（只读）
  • filterMode
    Filtering mode of the texture.
    贴图的过滤模式
  • anisoLevel
    Anisotropic filtering level of the texture.
    贴图的各向异性等级。（增加等级意味着更好的贴图效果，但是会需要更大的开销，减小则减少开销，代价是较远处或侧面的贴图会变得模糊。）
  • wrapMode
    Wrap mode (Repeat or Clamp) of the texture.
    循环模式 （重复或强制拉伸？）
  • mipMapBias
    Mip map bias of the texture.
    Mip贴图偏移
  • name
    The name of the object. //物体的名字
  • hideFlags
    Should the object be hidden, saved with the scene or modifiable by the user?
    物体是否被隐藏、保存在场景中或被用户修改？

  • ToString
