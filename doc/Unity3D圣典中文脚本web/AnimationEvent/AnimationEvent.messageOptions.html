AnimationEvent.messageOptions 消息选项

var messageOptions : SendMessageOptions

Description描述

Function call options.

函数调用选项。

If options are set to SendMessageOptions.RequireReceiver (default), an error is printed when the message is not picked up by any component.

如果选项设置为 SendMessageOptions.RequireReceiver (缺省),当消息没有被任何组件接收时打印一个错误信息.
