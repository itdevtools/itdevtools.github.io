Vector3.Project 投影

static function Project (vector : Vector3, onNormal : Vector3) : Vector3

Description描述

Projects a vector onto another vector.

投影一个向量到另一个向量。

Returns vector projected onto onNormal. Returns zero vector if onNormal is almost zero.

返回被投影到onNormal的vector。如果onNormal接近0，返回 0 vector。
