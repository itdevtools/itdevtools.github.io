ConfigurableJoint.xMotion X轴运动

var xMotion : ConfigurableJointMotion

Description描述

Allow movement along the X axis to be Free, completely Locked, or Limited according to Linear Limit

根据线性限制允许沿着X轴Free、完全Locked或Limited移动。
