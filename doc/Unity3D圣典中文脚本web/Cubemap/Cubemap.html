Cubemap 立方体贴图

Inherits from Texture

Class for handling cube maps, Use this to create or modify existing cube map assets.

用于处理立方体贴图的类，使用这个创建或修改现有的立方体贴图资源。

Variables变量

  • format
    The format of the pixel data in the texture (Read Only).
    在这个纹理像素数据的格式。（只读）

Constructors构造器

  • Cubemap
    Create a new empty cubemap texture.
    创建一个空的立方图纹理。

Functions函数

  • SetPixel
    Sets pixel color at coordinates (face, x, y).
    设置所在坐标(face, x, y)处的像素颜色。
  • GetPixel
    Returns pixel color at coordinates (face, x, y).
    返回所在坐标(face, x, y)处的样式颜色。
  • GetPixels
    Returns pixel colors of a cubemap face.
    设置一个Cubemap面的像素颜色。
  • SetPixels
    Sets pixel colors of a cubemap face.
    设置一个Cubemap面的像素颜色。
  • Apply
    Actually apply all previous SetPixel and SetPixels changes.
    实际应用所有先前SetPixel和SetPixels的改变。

Inherited members继承成员

Inherited Variables继承变量

  • width
    Width of the texture in pixels (Read Only)
    以像素为单位的贴图宽度 （只读）
  • height
    Height of the texture in pixels (Read Only)
    以像素为单位的贴图的高度（只读）
  • filterMode
    Filtering mode of the texture.
    贴图的过滤模式
  • anisoLevel
    Anisotropic filtering level of the texture.
    贴图的各向异性等级。（增加等级意味着更好的贴图效果，但是会需要更大的开销，减小则减少开销，代价是较远处或侧面的贴图会变得模糊。）
  • wrapMode
    Wrap mode (Repeat or Clamp) of the texture.
    循环模式 （重复或强制拉伸？）
  • mipMapBias
    Mip map bias of the texture.
    Mip贴图偏移
  • name
    The name of the object. //物体的名字
  • hideFlags
    Should the object be hidden, saved with the scene or modifiable by the user?
    物体是否被隐藏、保存在场景中或被用户修改？
