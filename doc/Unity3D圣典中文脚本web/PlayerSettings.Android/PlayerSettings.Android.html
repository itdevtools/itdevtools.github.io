PlayerSettings.Android 安卓播放器设置

Android specific player settings

安卓特定播放器设置。

Note: This is an editor class. To use it you have to place your script in Assets/Editor inside your project folder. Editor classes are in the UnityEditor namespace so for C# scripts you need to add "using UnityEditor;" at the beginning of the script.
注意：这是一个编辑器类，如果想使用它你需要把它放到工程目录下的Assets/Editor文件夹下。编辑器类在UnityEditor命名空间下。所以当使用C#脚本时，你需要在脚本前面加上 "using UnityEditor"引用。

Class Variables类变量

  • bundleVersionCode
    Android bundle version code // 安卓的软件包版本代码。
  • minSdkVersion
    Minimal Android SDK version // 最小的安卓SDK版本。
  • forceInternetPermission
    Force internet permission flag //强制网络权限标识。
  • forceSDCardPermission
    Force SD card permission // 强制SD卡权限。
  • targetDevice
    Android target device // 安卓目标设备。
  • targetGraphics
    Android target graphics level //安卓目标图形级别。
  • splashScreenScale
    Android splash screen scale mode // 安卓的启动画面缩放模式。
  • keystoreName
    Android keystore name // 安卓的的密钥名称。
  • keystorePass
    Android keystore password // 安卓的的密钥密码。
  • keyaliasName
    Android key alias name // 安卓的的别名名称。
  • keyaliasPass
    Android key alias password // 安卓的的别名密码。
  • licenseVerification
    License verification flag // 许可证验证标识。
