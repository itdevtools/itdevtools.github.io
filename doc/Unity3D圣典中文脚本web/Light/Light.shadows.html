Light.shadows 阴影

var shadows : LightShadows

Description描述

How this light casts shadows?

这个灯光怎样投射阴影？

参见：LightShadows, shadowStrength 属性，Renderer.castShadows, Renderer.receiveShadows.

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	void Awake() {
		light.shadows = LightShadows.Hard;
	}
}
// make the light cast hard shadows
//使该灯光投射硬阴影

light.shadows = LightShadows.Hard;
