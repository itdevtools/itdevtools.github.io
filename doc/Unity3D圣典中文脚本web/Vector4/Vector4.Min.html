Vector4.Min 最小

static function Min (lhs : Vector4, rhs : Vector4) : Vector4

Description描述

Returns a vector that is made from the smallest components of two vectors.

返回一个由两个向量的最小组件组成的向量。

参考：Max 函数。

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	public Vector4 a = new Vector4(1, 2, 3, 5);
	public Vector4 b = new Vector4(4, 3, 2, 0);
	public void Awake() {
		print(Vector4.Min(a, b));
	}
}
var a : Vector4 = Vector4 (1, 2, 3, 5);
var b : Vector4 = Vector4 (4, 3, 2, 0);
print (Vector4.Min(a,b)); 
// prints (1.0,2.0,2.0,0.0)
