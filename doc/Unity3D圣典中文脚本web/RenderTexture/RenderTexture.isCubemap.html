RenderTexture.isCubemap 是否立方图？

var isCubemap : bool

Description描述

If enabled, this Render Texture will be used as a Cubemap

如果启用，这个渲染纹理将使用Cubemap。

Cubemap render textures are most often used for dynamic cubemap reflections, see Camera.RenderToCubemap. A cubemap render texture must have the same width and height, and must be power of two size (see isPowerOfTwo).

Cubemap渲染纹理经常用于动态立方图反射，看Camera.RenderToCubemap。一个立方图渲染纹理必须有相同的宽度和高度，并且必须是2个乘方，看isPowerOfTwo。

参见： Camera.RenderToCubemap
