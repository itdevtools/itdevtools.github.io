Path 路径

Performs operations on strings that contain file or directory path information.

对包含文件或目录路径信息字符串执行操作。

This is the standard library System.IO.Path class. Documentation for frequently used class members is provided here for your convenience.

这是一个标准的System.IO.Path类库。经常使用的文档类成员为你提供方便。

Class Variables类变量

  • DirectorySeparatorChar
    The default character used to separate directory levels. (Read Only)
    默认字符用于分隔目录层次。（只读）
  • AltDirectorySeparatorChar
    An alternate character used to separate directory levels. (Read Only)
    一个替换字符用于分隔目录层次。 （只读）

Class Functions类函数

  • GetDirectoryName
    Returns the directory name component of the specified path string.
    返回指定路径字符串组件的目录名字。
  • GetExtension
    Returns the extension component of the specified path string.
    返回指定路径字符串的组件扩展名。
  • Combine
    Concatenates two path strings.
    连接两个路径字符串。
  • GetFileNameWithoutExtension
    Returns the file base component of the specified path string without the extension.
    返回指定路径字符串组件的基本文件名，不带有扩展名。
  • GetFileName
    Returns the file name, including the extension if any, of the specified path string.
    返回指定路径字符串的文件名，包含扩展名。
