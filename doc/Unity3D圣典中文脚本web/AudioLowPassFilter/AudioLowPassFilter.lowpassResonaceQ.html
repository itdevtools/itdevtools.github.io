AudioLowPassFilter.lowpassResonaceQ 低通共振质量

var lowpassResonaceQ : float

Description描述

Determines how much the filter's self-resonance isdampened.

定义了滤波器的自共振衰减多少。

Higher Lowpass Resonance Q indicates a lower rate of energy loss i.e. the oscillations die out more slowly.

较高的低通共振质量表示一个能量损失率较低，振动消失更慢。

Lowpass resonance Q value goes from 1.0 to 10.0. Default = 1.0.

低通共振质量值从1.0～10.0。默认=1.0。

  • C#
  • JavaScript
using UnityEngine;
using System.Collections;

public class example : MonoBehaviour {
	void Update() {
		GetComponent<AudioLowPassFilter>().lowpassResonaceQ = Mathf.Sin(Time.time) * 5 + 5;
	}
}
// Moves the Highpass Resonance Quality Factor from 0 to 10 following a Sinus function
// Attach this to an audio source with a LowPassFilter to listen it working.
//跟随Sinus函数移动高通共振质量系数从0～10，附加到一个带有LowPassFilter 的音频源
@script RequireComponent(AudioSource)
@script RequireComponent(AudioLowPassFilter)

function Update() {
	GetComponent(AudioLowPassFilter).lowpassResonaceQ = (Mathf.Sin(Time.time)*5 + 5);
}
