RectOffset 矩形偏移

Offsets for rectangles, borders, etc.

用于矩形，边框等的偏移。

Used all over the place by GUIStyle.

通过GUIStyle用于所有地方。

Variables变量

  • left
    Left edge size.
    左边缘大小。
  • right
    Right edge size.
    右边缘大小。
  • top
    Top edge size.
    顶边缘大小。
  • bottom
    Bottom edge size.
    底边缘大小。
  • horizontal
    shortcut for left + right (Read Only)
    left + right 简码。（只读）
  • vertical
    shortcut for top + bottom (Read Only)
    top + bottom 简码。（只读）

Constructors构造器

  • RectOffset
    Creates a new rectangle with offsets.
    创建一个新的矩形偏移。

Functions函数

  • Add
    Add the border offsets to a rect.
    向一个rect添加边框偏移。
  • Remove
    Remove the border offsets from a rect.
    从一个rect移除边框偏移。
