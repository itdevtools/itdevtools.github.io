FFTWindow.Blackman 布莱克曼窗

FFTWindow.Blackman

Description描述

w[n] = 0.42 - (0.5 * COS(n/N) ) + (0.08 * COS(2.0 * n/N) )

布莱克曼窗（三阶升余弦窗）。增加一个二次谐波余弦分量，可进一步降低旁瓣，但主瓣宽度进一步增加，增加N可减少过渡带。
最后修改：2011年5月9日 Monday 12:05

本脚本参考基于Unity 3.30f4
