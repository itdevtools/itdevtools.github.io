EventType 事件类型

Enumeration

Types of UnityGUI input and processing events.

UnityGUI输入的类型和处理事件。

参见： Event.type, Event

Values值

  • MouseDown
    Mouse button was pressed.
    鼠标键被按下。
  • MouseUp
    Mouse button was released.
    鼠标键被释放。
  • MouseMove
    Mouse was moved (editor views only).
    鼠标被移动（仅编辑器视图）。
  • MouseDrag
    Mouse was dragged.
    鼠标被拖拽。
  • KeyDown
    A keyboard key was pressed.
    一个键盘键被按下。
  • KeyUp
    A keyboard key was released.
    一个键盘键被释放。
  • ScrollWheel
    The scroll wheel was moved.
    滚轮被移动。
  • Repaint
    A repaint event. One is sent every frame.
    重复事件，是每帧发送。
  • Layout
    A layout event.
    一个布局事件。
  • DragUpdated
    Editor only: drag & drop operation updated.
    仅编辑器：拖放操作更新。
  • DragPerform
    Editor only: drag & drop operation performed.
    仅编辑器：拖放操作执行。
  • DragExited
    Editor only: drag & drop operation exited.
    仅编辑器：拖放操作退出。
  • Ignore
    Event should be ignored.
    事件应被忽略。
  • Used
    Already processed event.
    已经处理过事件。
  • ValidateCommand
    Validates a special command (e.g. copy & paste)
    验证一个特殊的命令（如，拷贝，粘贴）。
  • ExecuteCommand
    Execute a special command (eg. copy & paste)
    执行一个特殊的命令（例如，复制和粘贴）。
  • ContextClick
    User has right-clicked (or control-clicked on the mac).
    用户已右击（或mac上control-clicked）。
