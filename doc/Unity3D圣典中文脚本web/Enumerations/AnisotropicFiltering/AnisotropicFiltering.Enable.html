AnisotropicFiltering.Enable

Description描述

Enable anisotropic filtering, as set for each texture.

为每个纹理，启用各向异性过滤。

参见：QualitySettings.anisotropicFiltering.

最后修改：2011年4月26日 Tuesday 20:09

本脚本参考基于Unity 3.30f4
